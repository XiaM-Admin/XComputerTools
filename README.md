# XComputerTools
一个“最后开源”的C#电脑使用工具，快捷方便。\
其功能准备包括**账号本本、截图、Ocr、翻译、软件值守、自动化操作**等功能。\
最后开源：源码不开源，为了服务器的安全考虑。功能完善，程序稳定后，会将代码中自己的地址删除，并开源发布。\
软件不会收集任何用户信息，账号等，如果过度担心，请不要使用，感谢。
